⚙️ 首次設定 Firebase

為了讓資料在不同裝置之間同步，需要連接 Firebase 服務。請依照步驟填入你的 Firebase 設定（免費，5 分鐘完成）：

  1. 前往 console.firebase.google.com → 建立新專案
  2. 左側選單 → Authentication → 啟用 Google 登入
  3. 左側選單 → Realtime Database → 建立資料庫（選亞洲區域）
  4. 規則分頁貼上：{"rules":{".read":"auth!=null",".write":"auth!=null"}}
  5. 專案設定 ⚙️ → 你的應用程式 → 複製 firebaseConfig → 填入下方

設定只存在此瀏覽器，換電腦需重新填一次（或直接把填好的 HTML 檔案複製過去）
